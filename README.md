# Crop Recommendation System

This project recommends suitable crops based on soil nutrients and environmental conditions using Artificial Neural Networks (ANN).

## Technologies Used
- Python
- NumPy
- Pandas
- Scikit-learn
- Jupyter Notebook

## Project Files
- ipynb file (1).ipynb : Main notebook containing data preprocessing, ANN model training, and crop prediction.
